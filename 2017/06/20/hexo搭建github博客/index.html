hexo搭建github博客

2017-06-20

一直想搭建一个本地生成静态页面，然后放到线上，非常轻量级的博客。
最后找到了hexo搭配github的方案，连线上服务器都不需要，真的非常不错。
断断续续用了几天时间，终于算是完成了。

主要做了几件事记录一下：

  • 本地nodejs和hexo环境搭建
  • github域名仓库创建
  • 域名解析
  • 主题更换
  • google analytics
  • disqus
  • 添加备案信息
